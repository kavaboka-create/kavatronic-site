## 1. Цель проекта

### Проблема

Сайт вырос из лендинга smart-сервоприводов в точку входа для расширенной продуктовой линейки (5 функциональных ролей в мехатронном узле). Текущая структура не отражает платформенную модель и не масштабируется.

### Решение

Переработать главную страницу, навигацию и каталог под модель **мехатронной платформы из 5 согласованных линеек**, сохранив сервоприводы как якорный коммерческий продукт.

### Целевое позиционирование

**Kavatronic — промышленные компоненты для построения управляемых мехатронных узлов и мобильных систем: исполнение, измерение движения, вычисление, питание и восприятие среды.**

Тон: B2B, integrator/OEM, инженерный, без retail-витрины.

Сразу после `#platform`, **перед** `#servos-highlight`.

### Обязательные поля карточки

|------|------------|

2. IMU
3. Master Controller
4. Smart PDU
5. 2D/3D LiDAR / Depth Camera

### Value statements (рекомендуемые)

| Линейка | Value statement |
|---------|-----------------|

| Master Controller | Вычислительный модуль и шлюз данных для синхронизации узла |
| Smart PDU | Интеллектуальное питание, защита и телеметрия мехатронного узла |

### Поведение ссылок на P0

## 13. Технические ограничения

### Файлы к изменению на P0

| Файл | Действие |
|------|----------|
| `index.html` | Полная переработка структуры |
| `style.css` | Новые блоки, адаптация существующих |
| `script.js` | Только если нужно (удаление how-servo, безопасная очистка) |
| `products/*.html` | Обновление nav, CTA |
| `articles/*.html` | Обновление nav |
| `about.html` | Обновление nav |

### Что нельзя сломать (критично)

- `#contact-form` и Formspree submit
- `#kvt-success-modal`
- `#hamburger` и `#mobile-menu`
- `#contacts`
- Существующие product URLs
- Мобильная навигация

### Что можно удалить

С главной безопасно удалить:
- `#comparison`
- `#how-servo-works`
- `#servo-expand-btn`
- `#servo-detail`

**Условие:** без JS-ошибок после удаления.

### CSS

- Новые блоки: `.platform-scheme`, `.product-line-card`, `.servos-highlight` (или аналоги)
- Не удалять сразу `.products-subgroup` — deprecate после релиза
- Пересчитать порядок секций для полосатого фона `section:nth-of-type(even)`

### Constraints

- Статический HTML/CSS/JS
- Нет серверного кода
- Relative paths
- CDN для библиотек
- **Нет localStorage** (sandbox-блокировка)
- **Внешние ссылки:** `target="_blank" rel="noopener noreferrer"`
